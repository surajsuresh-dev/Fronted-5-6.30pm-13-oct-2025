UNITS

% -> percentage(screen resolution)

px -> pixel (1 pixel =1/96 inch)

mm -> millimeter

cm -> centimeter

pt -> point(1point = 1/72 inch)

ch -> width of the "0" glyph in the font size

em -> em,rem (1em =16px)(elements) (rem - responsive elements)

pc -> pica(1pica = 12points )

ex -> X-height of the element's font

gd -> grid defined by 'layout-grid'

vg & vw -> viewport height and width (current device resolution)
