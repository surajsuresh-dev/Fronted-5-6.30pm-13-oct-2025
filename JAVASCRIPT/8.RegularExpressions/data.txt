
Rules of RegularExpressions

1. /expression/ - if it is string

2. /^condition$/ 
        ^   -> open tag
        $   -> closed tag

3. []           - grouping (to group the regular expressions)

4. ()           - group of an mutiple charecter

5. [abcdef]     -input should match atleast one occurance of given charecters

6. [a-z] [A-Z] [0-9] [a-zA-Z0-9]

7. properties / operators
        .     => matches any charecter except line break
        ?     => match atleast 0 & 1
        *     => match 0 or more occurence
        +     => match 1 or more
        {n}   => matches exactly n occurence
        {n,m} => matches exactly n and m occurence

        g -> global
        gi ->global case insensitive
        u -> unicode
        y -> sticky
